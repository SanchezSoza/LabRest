Pagina de Inicio

Pagina de Inicio donde se puede navegar por tres botones, los cuales cada uno realiza una accion diferente y consume un sercivio Rest diferente, como muestran las cabeceras, estos servicios rest pueden mostrar todos los indicadores sin ningun parametro en especifico definido y hay otros en los cuales se solicitaran ingresar uno o dos datos para poder filtrar la informacion y entregar datos mas especificos
